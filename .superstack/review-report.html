Resolved high-impact findings

  • Public routing and validation no longer throw on prototype keys, accessors, revoked Proxies, ill-formed Unicode, or hostile error objects.
  • A render-ready payload is exact, bounded JSON and must satisfy the selected skill’s paired axes, numeric domains, units, identity, and provenance contract.
  • All five scene-less skills now use a strict host-renderer envelope instead of falling outside the honesty boundary.
  • NEST adapters preserve Float64 timestamps, reject Float32 overflow, retain measurement and spatial units, and refuse to merge distinct senders or synapses.
  • The React boundary is strict by default, memoizes validation, and gives each host render a detached snapshot so callback mutation cannot corrupt cached evidence.
  • Knowledge-graph layout is bounded, deterministic, refresh-independent, reduced-motion safe, and paired with paginated DOM semantics; population and neuron meshes have real DOM companions.
  • The generated manifest, declarations, runtime bundles, package exports, and documentation now ship together and are verified from a clean consumer.
